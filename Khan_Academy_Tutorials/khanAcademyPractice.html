!DOCTYPE html is required at the top of each html document. utf-8 is for the English alphabet. The title tag will appear in bookmarks and at the top of the tab. Browsers ignore whitespace, unless you use the br tag.

Break tags (br) do not need to be closed.

Br (break) breaks the line. It's a 'newline'.

Unordered List

  • ul stands for unordered list.
  • li stands for list item
  • wrap li tags inside ul tags
  • 'meta charset="utf-8"'' goes in the head

Numbered List

  1. ol stands for ordered list
  2. wrap li inside ol just like wrapping li in ul
  3. The strong tag makes all of the text inside the tag bold.
  4. The em tag makes a word italicised, as it emphasizes the word.

CSS Notes

  • Add the style tag to the head.
  • The style tag renders CSS.

  • You can change all of the paragraphs by using: p { color: rgb();}
  • Here's an example id:p id="paragraph1"
  • You can change just one paragraph by using an id, which is marked with a pound/hash symbol: #paragraph1 {color: rgb();}
  • IDs are case-sensitive. Use lower-case as to keep consistency.
  • IDs MUST be unique.

  • Classes are written with a period (.) instead of a pound/hash (#) like so: .lists {color: rgb();}
  • Classes are case-sensitive, just like IDs.

  • Don't put spaces in names of classes or IDs. Use hyphens or underscores.

CSS Examples

  • font-family: cursive;
  • font-family: "Helvetica", sans-serif;
  • font-size: 2em;
  • em is a relative unit, whereas px is not
  • font: italic 1.5em fantasy;

  • a tags mean anchor tags
  • href means hyper reference
  • target="_blank" opens the page in a new window
  • Example of opening a new page up: a target="_blank" href="url_goes_here"

  • Absolute URLs (full address) are safer than relative URLs (relative path)
  • You can jump to a different section on the same page by linking to an id.

Inserting an Image

The img src tag means 'image source'. Put an equals sign after to define the value. Wrap the value in quotes. Add the alt tag to give the image a title. Always use the alt tag with images as the blind use an app that can read the alt text off an image text. If you want everyone to experience your webpage, use the alt tag! Image tags do not need to be closed. Change either height or width, but not both, or else the proportions will be messed up!
